## 📖 درباره این مخزن (About This Repository)

**فارسی:**  
این مخزن شامل پروژه‌های عملی تحلیل داده است که من در طول مسیر یادگیری خود انجام داده‌ام. هدف از ساخت این مجموعه، نمایش توانایی‌های عملی در **پردازش، تحلیل و مصورسازی داده** با استفاده از زبان پایتون و کتابخانه‌های تخصصی آن است.

**English:**  
This repository contains practical data analysis projects I have completed during my learning journey. The goal is to demonstrate practical skills in **data processing, analysis, and visualization** using Python and its specialized libraries.

- ایجاد یک **پورتفولیوی حرفه‌ای** برای ارائه به کارفرمایان / Building a **professional portfolio** for employers

## 🚀 نحوه اجرا و استفاده (How to Use)

مراحل اجرا / Steps to run:

1. کلون کردن مخزن / Clone the repository:
   git clone https://github.com/0hasanbagheri0/Data-Analysis-Portfolio.git
   cd Data-Analysis-Portfolio

2. نصب کتابخانه‌های مورد نیاز / Install required packages:
   pip install pandas numpy matplotlib seaborn jupyter

3. اجرای نوت‌بوک یا اسکریپت مورد نظر / Run the desired notebook or script:
   jupyter notebook projects/project-01-titanic/analysis.ipynb
